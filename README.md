> Global Health & Development Dashboard
This project is an interactive Excel dashboard built using the WHO Life Expectancy dataset (2000–2015). It visualizes global health indicators and economic factors across countries.

> Key Features
Life Expectancy Trend (2000–2015)
KPIs: Average Life Expectancy, GDP (USD per capita), Infant Deaths, Health Expenditure
Developed vs Developing Comparison
Immunization Coverage (Polio, HepB, Measles, HIV/AIDS)
Alcohol Consumption vs GDP
Top 5 Countries by Infant Deaths
Interactive Filters: Year, Country, Status

> Tools Used
Microsoft Excel
Pivot Tables, Pivot Charts, Slicers

> Purpose
To explore how economic development and healthcare factors influence life expectancy across the world.
